# react-native-fraudforce

React Native wrapper for FraudForce sdk (aka iovation)

## Installation

```sh
npm install react-native-fraudforce
```

## Usage

```js
import { multiply } from "react-native-fraudforce";

// ...

const result = await multiply(3, 7);
```
